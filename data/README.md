# Data Directory

## MALDI_positions.csv

Contain the corresponding relation of samples and their position on MALDI plates.

## MALDI_raw

The raw MALDI data files.

## mass_list

Mass list xlsx file exported from flexAnalysis.
Each experimental plate has one file.

## plates.csv

The position of each sample, and its two sample names,
one for the raw sample names given by the hospital,
the other for the sample names used in this project.

## human_serum_glycans.csv

The glycoCT structure strings of human serum glycans.
